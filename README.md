> **Neural Sequencing for Technical Identity Synthesis.**

**DNA.dev** is a premium, high-fidelity GitHub profile analyzer designed to transform raw repository data into a cinematic technical narrative. It uses advanced neural-inspired visualization and AI-driven classification to decode a developer's career trajectory.

---

## 🛠️ Neural Core Features

### 🧠 AI Archetype Detection
The platform sequences your technical DNA to categorize you into professional archetypes. Are you an **Interface Architect**, a **Systems Core Designer**, or a **Neural Operations Engineer**? Our data-aware engine decides based on your real output.

### 🗓️ Annual Contribution Flux
A high-fidelity 365-day activity grid that visualizes your "Neural Flux." Using a seeded simulation engine, it reflects the intensity and frequency of your repository updates in a stunning primary-accented heatmap.

### 🔬 Technical DNA Report
Extract deep telemetry of your projects. The platform provides a functional export tool to download your analyzed DNA report as a JSON payload for external validation.

### 🔐 DNA Vault (Auth System)
A professional multi-user registry with persistent session management. Secure your identity within the local lab and track your authorization history.

## 🚀 Initialization

To boot up the Neural Sequencing Lab:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Souharda6996/Github-Portfolio-Manager.git
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env.local` file for deep AI analysis (optional):
   ```env
   OPENAI_API_KEY=your_key_here
   ```

4. **Launch the Engine:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to begin the sequencing.
